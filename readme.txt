  테트리스 (Tetris) - 간단·상세 매뉴얼

■ 실행 방법
--------------------------------------------------------------------------------
  1. index.html 파일을 더블클릭하거나 브라우저에서 열기
  2. 별도 설치 없이 바로 플레이 가능 (인터넷 연결 필요: 폰트·Supabase 리더보드)


================================================================================
  [간단 매뉴얼]
================================================================================

■ 시작 / 다시 시작
  - 대기 화면: "시작" 버튼 클릭 또는 스페이스
  - 게임 오버 후: 1초 뒤 이름 입력 → "다시 시작" 버튼 또는 스페이스

■ 조작 (키보드)
  ← →   좌우 이동
  ↓     한 칸 아래 (소프트 드롭)
  ↑     회전 (벽에 걸리면 자동으로 좌우 밀어서 회전 시도)
  스페이스  즉시 바닥까지 낙하 (착지 후 좌/우로만 한 칸씩 이동 가능, 0.5초 뒤 락)
  P      일시정지

■ 화면 표시
  - 낙하 속도(%) : 점수가 올라갈수록 자동 낙하가 빨라짐 (10점당 2% 증가)
  - 점수, 레벨, 라인 : 상단에 표시
  - 좌측 : 최고 기록 TOP 10 (Supabase 연동)
  - 우측 : 다음 블록 미리보기 + 조작법

■ 자신의 기록
  - 순위표에서 내 기록의 순위 숫자는 보라색
  - 1·2·3위인 내 기록은 갈색으로 표시


================================================================================
  [상세 매뉴얼]
================================================================================

■ 1. 게임 규칙
--------------------------------------------------------------------------------
  - 7가지 블록(I,O,T,S,Z,J,L)이 위에서 떨어짐
  - 가로 한 줄이 꽉 차면 그 줄이 사라지고 점수·레벨 상승
  - 새 블록이 스폰될 자리가 막혀 있으면 게임 오버

■ 2. 점수
--------------------------------------------------------------------------------
  - 라인 클리어
      1줄: 100 × 레벨
      2줄: 300 × 레벨
      3줄: 500 × 레벨
      4줄: 800 × 레벨
  - 소프트 드롭(↓ 한 칸): +10점
  - 즉시 낙하: (줄 수 × 2 × 즉시낙하 배율) 점 (배율은 낙하 속도의 20%만 반영)

■ 3. 레벨·속도
--------------------------------------------------------------------------------
  - 레벨: 쌓인 라인 10줄마다 1레벨 상승
  - 낙하 속도(%): 100%에서 시작, 10점당 2% 증가 (표시만, 자동 낙하에만 적용)
  - 자동 낙하: 레벨이 올라갈수록 기본 간격이 짧아지고, 점수에 따른 속도 %로 더 짧아짐
  - 즉시 낙하(스페이스): 위 속도 증가의 20%만 반영 (예: 300%일 때 140% 배율)

■ 4. 즉시 낙하(스페이스) 상세
--------------------------------------------------------------------------------
  - 스페이스 한 번: 블록이 바닥까지 즉시 낙하
  - 착지 후 약 0.5초 동안:
      · 좌(←) 우(→)로만 이동 가능 (각각 최대 1칸씩)
      · 스페이스 추가 입력 불가
      · 아래로 떨어질 공간이 생기면 락하지 않고 다시 일반 낙하로 전환
  - 0.5초 경과 시 락(고정) → 다음 블록

■ 5. 회전(월킥)
--------------------------------------------------------------------------------
  - ↑ 키로 회전. 벽이나 블록과 겹치면 자동으로 좌우로 밀어서 회전 시도
  - 대부분의 경우 회전이 성공하도록 설계됨

■ 6. 라인 클리어 연출
--------------------------------------------------------------------------------
  - 한 줄 이상 완성 시 해당 줄이 즉시 사라지지 않고 약 0.48초 동안 리드미컬한
    플래시 연출 후 한꺼번에 제거됨

■ 7. 최고 기록(리더보드)
--------------------------------------------------------------------------------
  - Supabase에 player_id(쿠키) + 이름 + 점수 저장
  - 좌측 패널에 상위 10개 표시
  - 자신의 기록: 순위 숫자 보라색, 1·2·3위면 갈색
  - 게임 오버 후 1초 뒤 이름 입력창 → 입력 시 자동 저장·갱신

■ 8. 설정 파일 (고급)
--------------------------------------------------------------------------------
  - config.js : Supabase URL, anon key (다른 프로젝트 사용 시 수정)
  - Supabase 테이블: supabase/schema.sql 내용을 Supabase 대시보드 SQL Editor에서
    한 번 실행해야 리더보드 저장·조회 가능


================================================================================
  [문제 해결]
================================================================================

  - 기록이 안 남아요
    → Supabase 대시보드에서 supabase/schema.sql 실행 여부 확인
    → config.js의 URL·anon key가 해당 프로젝트와 일치하는지 확인

  - 스코어보드가 "불러오기 실패"
    → 인터넷 연결 확인
    → Supabase 프로젝트가 활성화되어 있는지, RLS 정책이 적용되었는지 확인

  - 쿠키를 지우면?
    → 새 플레이어 ID가 부여되어 이전 기록과 다른 "플레이어"로 인식됨
      (이름이 같아도 순위표에서 본인 기록 강조가 달라질 수 있음)
